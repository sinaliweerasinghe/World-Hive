# 🌍 World Hive

World Hive is a responsive web platform developed as a group project to provide lifestyle content, book reviews, and informative resources through an engaging user experience.

## ✨ Features

- Responsive Web Design
- Lifestyle Content Section
- Book Review Platform
- API Integration
- JSON Data Handling
- Mobile-Friendly Interface
